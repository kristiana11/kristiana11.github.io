Projects

1. Golf-N-Stuff

This project was developed for our Software Engineering Class. Our team decided to develop a minature golf game which could be played from any computer browser that was connected to the enternet. In order to do this, we utalized Unity, itch.io, and some free resource packages that we found online. Through the semester, we were able to accomplish it. In the team, we had a manager, quality assurance, and the team. I was the quality assurance person, which meant I reviewed the pull requests and pulled it to the main branch. I also was in charge of submitting our Deliverables and making sure everyone was on top of their tasks. I also helped with some UI and terrain for the game. I had never touched Unity before, so this project was a stuggle at first but I had a great team that helped me through learning Unity. The main thing I learned from this project is communcation is key. We struggled at this during the beginning of the project because we did not know each other and did not want to step on other peoples toes. Towards the end, we were able to ask each other for help and see where our project was going. Overall, this project taught me how to work in a team enviroment and really use the agile methodology we learned in class.

2. Bookify

This project was developed for my Web Development II Class. The purpose of Bookify is to provide an online platform for users to explore, purchase, and read books convenietly. We based this project off of Kindle/Apple Books. Bookify has a user-friendly interface that encourages user to browse books, search/discovery, user registration, shopping cart and checkout, and users should be able to manage their profile. In this project, I took on the role of team lead along with developer since it was a small project. I developed the search bar, shopping cart/checkout, and the styling the books and website. I had not used React before, so we struggled with putting it onto GitHub as well as update it each time. This project used React and Node JS, along with the understanding of API and mongoDB. This project helped me learn more about how to use mongoDB, Node JS, React, and implement the database to get information. Overall, this project taught me how to use my agile methods I attained from Software Engineering and learned how to make the foundation of this website using a new language.
